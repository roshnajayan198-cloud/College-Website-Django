# 🎓 College Website

A modern and responsive college website developed using Django.  
This project provides an online platform to showcase college information, facilities, events, gallery, and contact details.

The website is designed to provide a simple and user-friendly experience for students, visitors, and applicants.

## 🚀 Features

- 🏫 College information and overview
- 💼 Placement support details
- 🎉 Events and gallery section
- 📩 Contact form with database storage
- ✅ Success message after form submission
- 📱 Responsive design for different devices

## 🛠️ Technologies Used

- **Frontend**
  - HTML
  - CSS
  - JavaScript

- **Backend**
  - Python
  - Django

- **Database**
  - SQLite

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/your-username/College-Website-Django.git
Move into project folder
cd College-Website-Django
Create virtual environment
python -m venv venv
Activate virtual environment

Windows:

venv\Scripts\activate
Install dependencies
pip install -r requirements.txt
Apply migrations
python manage.py migrate
Run the development server
python manage.py runserver

Open in browser:

http://127.0.0.1:8000/
